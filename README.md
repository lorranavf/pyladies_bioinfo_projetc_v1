# Description

A Python package for manipulating biological sequences and handling FASTA file I/O.

## Install

```bash
git clone git@github.com:lorranavf/bioseq.git
```


## How to Use

To learn how to utilize the package features, refer to the `examples` folder. It contains Jupyter Notebooks demonstrating key functionalities, such as reading FASTA files and manipulating sequences.

## Running Tests

Run the following command to execute the test suite and verify the package functionality. Note that make is required.

```bash
make test
```
