# Patterns

## Creational Patterns

### Singleton

Usado para criar objetos com uma única instância.

### Prototype

Usado para criar clones de objetos sem depender de suas respectivas classes.

### Factory Method

Permite a  criação de objetos a partir de superclasses, mas de modo que subclasses possam alterar o tipo do objeto.

### Abstract Factory

Permite a criação de objetos a partir de superclasses sem a necessidade de especificar suas classes concretas.

### Builder

Permite a criação de objetos passo-a-passo
